## 👥 회원 권한 구조

| 등급 | 전환 방법 | 접근 범위 |
|------|-----------|-----------|
| 비회원 | — | 공개 정보, 공지사항, 교육 안내 |
| 일반회원 | 본인인증 후 직접 가입 | 비회원 + 자유게시판, 일자리정보, 강좌 신청 |
| 정회원 | **관리자 수동 등급 변경** (자동화 없음) | 회원 + 역량강화·사공단·동아리 게시판 |
| 관리자 | 시스템 설정 | 전체 |

> ⚠️ 정회원 자동 전환 없음. 교육 수료 + 회비 납부 확인 후 관리자가 수동 처리.
> ⚠️ 회원가입 시 휴대폰 본인인증 필수 (NICE 또는 이니시스, 건당 비용 발생).

## ⭐ 핵심 기능 명세

### 1. 협회 일정 캘린더

- 월별 뷰 + 목록형 뷰 전환
- 일정 클릭 시 관련 게시물/URL로 이동
- 링크 없는 단순 텍스트 일정도 지원
- 관리자: 목록 형태로 등록/수정/삭제
- 일정별 권한 처리 가능
- 카테고리: 교육/강좌, 사회공헌 활동, 협회 회의, 동아리 모임, 행사
- 열람: 전체 공개 / 등록·수정·삭제: 관리자 전용

### 2. 강좌 통합 신청 시스템

**적용 강좌**
- 숲해설가 전문가과정
- 시민아카데미 교육강좌
- 회원활동 역량강화 교육강좌
- 숲해설 강사 신청 (결과 공지 없음, 신청만 수집)
- 후원하기 신청폼

**강좌 개설 필드 (관리자)**
: 강좌명 / 교육 기간(시작~종료) / 기수 / 정원 / 카테고리 / 신청 가능 여부

**신청 흐름 (사용자)**
```
신청 안내 페이지
  → 개인정보 동의 (체크박스)
  → 신청 폼 (이름/연락처/이메일/지역/파일업로드/기타)
  → 신청 완료 페이지
```

**신청자 관리 (관리자)**
: 강좌별 신청자 목록 / 상태 변경(신청완료·수료·취소) / 엑셀 다운로드 / 수료증 PDF 개별 다운로드

### 3. 수료증 PDF 발급

**수료증 포함 항목**: 성명 / 과정명 / 교육 기간 / 기수 / 발급일 / 발급 기관(고정값)

**처리 흐름**
```
관리자: 신청자 상태 → '수료'로 변경
  ↓
사용자 마이페이지 → 교육이수내역 → '수료증 다운로드' 버튼 활성화
  ↓
지정 포맷으로 PDF 생성 → 다운로드
```

> ⚠️ 기존 수료증은 HWP 형태. 고객사로부터 HWP 샘플 + 실물 이미지 수령 후 디자인 확정 필요.

## 🗂️ 게시판 유형 정책

| 유형 | 설명 | 해당 게시판 |
|------|------|------------|
| 정보P | 관리자가 퍼블리싱으로 작성, CMS 관리 없음 | 인사말, 미션, 연혁, 정관 등 |
| 게시판P | 에디터 사용, 글/사진 등록 가능 | 공지사항, 동아리소식, 사공단소식 등 |
| 기능P | 신청폼, 캘린더 등 개발 필요 | 교육신청, 협회일정, 수료증 발급 등 |

- 댓글 허용 게시판 목록: **미확정 (고객사 확인 필요)**
- 파일 업로드 용량 제한 필수 (사공단 일지 등)
- 허용 확장자: JPG, PNG, PDF, HWP, DOCX (개발 시 확정)

### 게시물 삭제 권한 정책 ✅ 2026-03-19 확정

> 별도 관리자 전용 게시물 관리 페이지 없음. 상세 페이지 인라인 처리 방식 채택.

| 대상 | 삭제 가능 범위 |
|------|--------------|
| 작성자 본인 | 자신이 작성한 글만 |
| 관리자 | 전체 게시판 모든 글 |
| 비회원·일반회원·정회원 | 타인 글 삭제 불가 |

**구현 방식**
- 게시물 상세 페이지에서 로그인 상태 + 권한 체크 후 "삭제" 버튼 조건부 노출
- `App.user.role === 'admin'` 또는 `post.authorId === App.user.id` 일 때 버튼 표시
- 삭제 처리: **소프트 삭제** (`deleted_at` 필드 기록, 실제 row 제거 금지)
- 삭제 후 목록 페이지로 리디렉션 + 토스트 메시지 표시
- 수정 버튼도 동일 조건 적용 (작성자 본인 + 관리자만 노출)

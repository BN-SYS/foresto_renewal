# (사)한국숲해설가협회 홈페이지 전체 재구성 가이드

> **문서 기준**: 요구사항정의서 V2.0 (2026-03-18)  
> **목적**: 기존 소스 전면 개편 — 페이지 구조·기능·접근권한 기준 재정립  
> **대상**: Claude Code (전체 소스 재구성 작업 지시서)

---

## 0. 작업 원칙

1. **기존 소스 재활용** — 완전 삭제 후 신규 생성이 아닌, 기존 파일 구조 기반으로 재편
2. **페이지 단위 완성** — 한 페이지 완성 후 다음 페이지로 진행 (CSS 깨짐 방지)
3. **공통 컴포넌트 우선** — Header / Footer / Sidebar / Layout 먼저 확정 후 각 페이지 적용
4. **접근권한 반드시 구현** — 비회원 / 회원 / 정회원 / 관리자 4단계 권한별 라우팅 제어
5. **신규 페이지 누락 없음** — 아래 전체 페이지 목록 기준으로 빠짐없이 생성

| 항목 | 내용 |
|------|------|

## 3. 접근권한 매트릭스

| 구분 | 권한레벨 | 접근 가능 메뉴 |
|------|---------|--------------|
| 비회원 | 0 | 메인, 소개 전체, 교육과정 소개/신청, 동아리 소개, 커뮤니티(자료실 제외), 숲해설 신청, 회원가입·로그인 |
| 회원 | 1 | 비회원 + 수료생 후기 글쓰기, 강사 신청, 마이페이지, 자료실 열람 불가 |
| 정회원 | 2 | 회원 + 회원활동 전체(아카데미·사공단·동아리소식·자료방), 커뮤니티 자료실 |
| 관리자 | 9 | 전체 + 관리자 페이지 |

**구현 방식:**
- 로그인 상태 + 권한 레벨 체크 미들웨어
- 미인가 접근 시 로그인 페이지로 리다이렉트 (권한 부족 시 안내 메시지 표시)
- 정회원 자동 전환 없음 — 관리자 수동 설정

## 4. 공통 컴포넌트 명세

### 4-1. Header

```
[로고] [GNB 메뉴] [글자크기A+A-] [로그인/회원가입 or 마이페이지/로그아웃]
```

- 반응형: PC(GNB 풀노출) / 모바일(햄버거 메뉴)
- 글자 확대·축소 버튼: 전역 font-size 제어
- 로그인 상태에 따라 우측 버튼 영역 변경

### 4-2. Footer

```
[협회 정보 (로고, 주소, 연락처, 사업자번호)]
[구 홈페이지 링크]
[유관기관 링크 - 좌우 슬라이드]
[이용약관 | 개인정보취급방침 | 이메일 무단수집거부]
[Copyright]
```

### 4-3. 서브페이지 공통 레이아웃

```
[헤더]
[페이지 타이틀 + 브레드크럼 (현재위치)]
  └── 좌측: LNB (서브메뉴)
  └── 우측: 콘텐츠 영역
[푸터]
```

### 4-4. 게시판 공통 컴포넌트

- 목록형: 제목 / 작성자 / 날짜 / 조회수 + 페이지네이션
- 갤러리형: 썸네일 이미지 그리드
- 상세: 제목 / 내용 / 첨부파일 / 목록으로 버튼
- 검색: 제목+내용 통합 검색
- 글쓰기 권한 분리 (접근권한별)

### 4-5. 신청 폼 통합 컴포넌트 (공통)

**공통 필드:**
- 이름 (필수)
- 연락처 (필수, 010-xxxx-xxxx 형식)
- 이메일 (필수)
- 개인정보수집이용 동의 체크박스 (필수)
- 제출 버튼

**과정별 추가 필드:**
- 교육 신청: 지역 선택, 파일 업로드
- 강사 신청: 활동 경력, 첨부파일
- 후원 신청: 후원 금액, 내용
- 숲해설 신청: 기관명, 신청 내용, 방문 희망일

## 8. CSS / 디자인 재구성 원칙

1. **공통 CSS 변수 정의 우선**
   ```css
   :root {
     --color-primary: [협회 그린 계열];
     --color-secondary: [보조 색상];
     --font-base: 16px;
     --font-large: 18px; /* 글자 확대 시 */
     --font-xlarge: 20px; /* 최대 확대 */
   }
   ```

2. **컴포넌트 CSS 분리**: 헤더 / 푸터 / 게시판 / 폼 / 캘린더 각각 독립 파일

3. **레이아웃 그리드**: PC 최대 1200px 컨테이너 / 서브페이지 좌측 LNB 220px + 우측 콘텐츠

4. **반응형 브레이크포인트**:
   - PC: 1200px 이상
   - 태블릿: 768px ~ 1199px
   - 모바일: 360px ~ 767px
